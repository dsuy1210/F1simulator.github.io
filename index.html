Mô phỏng đua F1 với luật 2026.

Repo này hiện chủ yếu là mô phỏng bằng Python (lap time model, tyre degradation, pit stop, safety car, weather, active aero…). GitHub Pages chỉ host được trang tĩnh, nên đây là landing page để giới thiệu & trỏ link vào source.

Quick start (local)

Nếu bạn muốn chạy mô phỏng trên máy (Python), dùng các lệnh sau từ thư mục ppj-f1-simulator. 

cd ppj-f1-simulator
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python -m f1_simulator.main

GitHub Pages

Để Pages hiển thị đúng, chỉ cần có file index.html ở root (chính là trang này). Nếu bạn muốn thay thế bằng web app thật, hãy build ra static files và để ở root hoặc cấu hình Pages trỏ tới thư mục build.

Mô hình lap time

Công thức mục tiêu (tóm tắt theo README):

LapTime =
BaseLap
+ TyreDelta
 + (TyreDeg × TyreAge)
+ TrackConditionModifier
- ActiveAeroBonus
- OvertakeModeBonus
- PitOutTyreBonus

Safety Car

  • Tăng lap time ~ 40%
  • Gaps shrink còn ~ 30%
  • Pit dưới SC giảm pit loss ~ 10s

Luật 2026

  • Không còn DRS
  • Active Aero (low-drag trên straight)
  • Overtake Mode khi gap ≤ 1s
  • Power unit ~ 50% điện, ERS recharge tăng
